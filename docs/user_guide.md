# MIA Search User Guide (MVP v0.1.0)

## 1. Установка и запуск

### Вариант A: готовый Windows installer
1. Запустите `src-tauri/target/release/bundle/nsis/MIA Search_0.1.0_x64-setup.exe`.
2. Установите приложение стандартным wizard.
3. Откройте `MIA Search` из меню Пуск.

### Вариант B: запуск из исходников (dev)
```bash
npm install
npm run dev
cd src-tauri
cargo tauri dev
```

## 2. Базовый сценарий поиска
1. Введите запрос в верхнем поле.
2. Выберите режим кнопками под строкой поиска: `PLAIN`, `WC`, `RE`.
3. При необходимости добавьте root-папки через левую панель.
4. Нажмите поиск (или используйте live-search).
5. Результаты приходят потоково, первые записи появляются до окончания полного сканирования.

Примечание: режимы и флаги поиска теперь переключаются UI-кнопками/переключателями, команды в поле ввода писать не нужно.

## 3. Фильтры и сортировка

Доступные фильтры:
- Тип: все / только файлы / только папки.
- Расширения: одно или несколько.
- Размер: `greater/smaller/equal`.
- Дата: `created/modified before/after`.

Сортировки:
- `relevance`
- `name`
- `size`
- `modified`
- `type`

## 4. Действия над результатом

Для выбранной записи доступны:
- `Open`
- `Open parent`
- `Reveal in Explorer`
- `Copy path` / `Copy name`

## 5. История, избранное, профили

- История сохраняет поисковые запросы и открытые пути.
- Избранное хранит закреплённые пути.
- Профили позволяют сохранять полный набор параметров поиска.
- Настройки, история, избранное и профили сохраняются между запусками.

## 6. Полезные заметки

- Поиск можно отменять: при новом live-запросе предыдущий останавливается автоматически.
- По умолчанию backend поиска: `Scan`.
- В настройках поиска можно включить `Index`, посмотреть статус индекса и вручную запустить `Rebuild index`.
- Для `Index` доступна авто-реиндексация по TTL и интервалу проверки (настраивается в Settings).
- Горячая клавиша ручной реиндексации: `Ctrl/Cmd + Shift + R`.
- Для стабильной производительности на больших деревьях ограничивайте `depth` и `limit`.
